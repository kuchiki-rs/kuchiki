fn from_http<R: IntoResponse>(self, response: R) -> Result<NodeRef>

Fetch an HTTP or HTTPS URL with Hyper and parse, giving the charset parameter of a Content-Type response header, if any, as a character encoding hint to html5ever.

This feature (and the dependency on Hyper) is optional and not enabled by default. To enable it, your Cargo.toml file should look like:

[dependencies]
kuchiki = {version = "...", features = ["hyper"]}

See Cargo features.
